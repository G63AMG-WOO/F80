# 운동 수행 능력 향상을 위한 청각적 격려 프로그램

##주요 기능
- **청각적 격려 출력**: 반복 횟수를 세어주고 사용자의 움직임이 느려질 시 '할 수 있어' 와 같은 음성을 출력합니다.
- **운동 통계 제공**: 반복 횟수를 자동으로 저장하고 이를 바탕으로 시각적 운동 통계를 제공합니다.
